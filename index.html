🎯 Problema escolhido & Necessidades

📌 Problema / Necessidade

Muitas pessoas têm dificuldade em controlar, de forma simples e rápida, os gastos do dia a dia, resultando em descontrole financeiro e falta de visibilidade sobre para onde o dinheiro está indo.

✔ Solução proposta: Aplicativo mobile "Meu Bolso em Dia" que permite registrar despesas diárias com poucos toques, categorizar e acompanhar resumos.

✅ Requisitos Funcionais

  • RF01 - Registrar despesas diárias (data, tipo, descrição, valor, forma de pagamento).
  • RF02 - Listar todas as despesas ordenadas por data (mais recentes primeiro).
  • RF03 - Visualizar resumo de gastos por categoria (alimentação, transporte, saúde, lazer).
  • RF04 - Editar ou excluir despesas registradas.
  • RF05 - Exibir gasto total do mês atual.

⚙️ Requisitos Não-Funcionais

  • RNF01 - Aplicativo mobile (Android/iOS) com interface intuitiva.
  • RNF02 - Armazenamento local (SQLite/Realm) para segurança e privacidade.
  • RNF03 - Tempo de resposta < 1 segundo para operações principais.
  • RNF04 - Design acessível e com modo claro/escuro (opcional).

🔄 Modelo de Procedimento
1
Registrar despesa
Usuário acessa tela "Nova Despesa" → preenche data, tipo, descrição, valor e forma de pagamento → sistema valida campos (valor >0, tipo válido) → armazena localmente no banco de dados → atualiza lista e resumo.
2
Visualizar despesas
Ao iniciar ou clicar em "Início", o sistema exibe lista com últimas despesas e total do mês atual. Caso deseje filtrar por categoria → seleciona o tipo no menu de filtros → aplica query no modelo de dados.
3
Editar / Excluir
Na lista de despesas, usuário desliza sobre um item (gesto swipe) ou clica no ícone de edição → altera dados da despesa ou confirma exclusão → sistema atualiza o registro no modelo de dados e recalcula totais automaticamente.
4
Relatório por período
O usuário seleciona "Relatórios" → escolhe mês/ano → sistema consulta a tabela 'Despesa' agrupando por tipo e devolve resumo com gráfico (procedimento de agregação). Baseado nos atributos data e valor.
🔁 Procedimento em pseudocódigo (registro):
function registrarDespesa(data, tipo, desc, valor, pagamento):
  if validarCampos() and valor > 0:
    INSERT INTO Despesa VALUES (...);
    atualizarTotalMensal();
    return sucesso;
  else: exibirErro("Dados inválidos");
